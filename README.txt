TRAZO - Web funcional final
===========================

Esta carpeta contiene una web estática completa para la tienda online de TRAZO.

ARCHIVOS PRINCIPALES
- index.html: home, presentación de marca, valores, productos, opiniones y contacto.
- product.html: ficha individual de producto, con selector de talla y galería look/producto.
- cart.html: cesta funcional y checkout simulado.
- styles.css: estética, responsive y todos los estilos.
- products.js: base de datos completa de los 23 productos.
- app.js: interactividad, filtros, galería, carrito y checkout.

CARPETAS
- assets/brand: logo y etiqueta de marca.
- assets/looks: looks completos, nombrados look-01.webp a look-23.webp.
- assets/products: fotos individuales, nombradas product-01.webp a product-23.webp.

CÓMO PROBARLA
1. Abre esta carpeta en VS Code.
2. Haz clic derecho sobre index.html.
3. Elige "Open with Live Server".
4. Navega por productos, prueba product.html?id=01, añade tallas a la cesta y revisa cart.html.

NOTAS
- La correspondencia de imágenes respeta el orden pedido: look-01 va con product-01, look-02 con product-02, etc.
- El pago no es real; el checkout solo simula el final del proceso.
- La web no usa frameworks ni dependencias externas.
- La estética sigue el manual de identidad: Baskerville, Buttermilk #FFF1B5, Olive Line #7B844A, Pastel Blue #C1DBE8 y Old Burgundy #43302E.
