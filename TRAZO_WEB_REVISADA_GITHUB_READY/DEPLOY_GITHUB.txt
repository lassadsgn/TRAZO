TRAZO · INSTRUCCIONES PARA GITHUB PAGES
=======================================

Para que no salga 404 en GitHub Pages, en la raíz del repositorio deben verse directamente estos archivos:

index.html
product.html
cart.html
styles.css
products.js
app.js
assets/
.nojekyll

NO subas solo el ZIP a GitHub.
NO dejes todo dentro de una carpeta tipo TRAZO_web_REVISADA_GITHUB si Pages publica desde /root.

Pasos:
1. Descomprime este ZIP.
2. En GitHub, entra en el repositorio.
3. Sube TODO el contenido de esta carpeta a la raíz del repositorio.
4. Comprueba que index.html se ve directamente en la pestaña Code.
5. Ve a Settings > Pages y deja: main / (root).
6. Espera unos minutos y abre la URL pública.
